рҹ”җ Security Checklist

Passwords-
bcrypt (12 salt rounds)
Never store plain text passwords

JWT-
Use two tokens:
Access Token
Expires in 15 minutes
Sent with API requests
Refresh Token
Expires in 7 days
Stored securely
Used to issue new access tokens

Cookies-
Store the refresh token in an HttpOnly, Secure cookie so JavaScript cannot read it. This helps protect against token theft through XSS.

Validation-
Validate every request:
Name
Email
Password
File uploads

Never trust frontend validation alone.
